# Itinera
Web - Based Travel Planning Tool 
 
Itinera is a simple web app built to make travel planning easier by keeping everything in one place. Instead of juggling multiple apps or notes, users can organize their itinerary, track packing lists, and manage expenses all in a single interface.

The project is built using HTML, CSS, and JavaScript, with a focus on clean design and easy-to-use features. It’s deployed with GitHub Pages and serves as a practical front-end project that demonstrates core web development skills.
